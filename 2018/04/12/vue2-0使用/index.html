vue2.0开始走起

前端开发 2018/04/12

本文于382天之前发表，文中内容可能已经过时。

前置知识

1.html、css、javascript基础知识需要一定的掌握
2.node.js初级知识，需要会npm的使用和项目初始化

下载Vue2.0的两个版本

官方网站：https://vuejs.org/

  • 开发版本：包含完整的警告和调试模式（vue.js）
  • 生产版本：删除了警告，进行了压缩（vue.min.js）
项目结构搭建

新建项目结构如下图（只是简单新建，真正项目中会比较复杂）：


1.新建vue2.0文件夹，在此文件夹中分别新建assets文件（主要用于存放静态文件）、example文件以及index.html
2.assets文件夹下新建css文件夹、js文件夹

live-server的使用

安装环境:node.js npm
live-server是一个简单级的开发服务器

cnpm install -g live-server   全局安装

live-server  在项目目录打开

将vue.js放入 assets下的js文件夹

编写第一个HelloWorld代码：
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <script type="text/javascript" src="../assets/js/vue.js"></script>
    <title>Helloworld</title>
</head>
<body>


    <div id="app">
        {{message}}
    </div>

    <script type="text/javascript">
        var app=new Vue({
            el:'#app',
            data:{
                message:'hello Vue!'
            }
        })
    </script>
</body>
</html>

为方便以后进行包的管理，此刻最好先 npm init项目初始化，根据提示一步步完成package.json
